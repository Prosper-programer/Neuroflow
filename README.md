# 🧠 NeuroFlow - AI Powered Productivity

**NeuroFlow** is a premium, high-performance landing page designed for an AI-driven productivity platform. It features a modern, dark-themed aesthetic with advanced CSS animations, glassmorphism, and a focus on interactive user experience.

---

## ✨ Key Features

- **🚀 Interactive Hero Section**: Futuristic AI-themed visuals with fluid typography and a floating animation system.
- **🛠️ Zigzag Process Timeline**: A unique "How It Works" layout that maintains its alternating zigzag flow across both desktop and mobile devices.
- **📱 Premium Responsive Navigation**: A custom-built mobile menu featuring a "hamburger-to-cross" animation and high-blur glassmorphic overlays.
- **💬 Focused Testimonial Carousel**: A 1-at-a-time testimonial slider with an asymmetric dash-style progress bar and floating side-navigation.
- **📊 Showcase Dashboard**: A side-by-side platform preview section that highlights key features with interactive hover states.
- **✨ Scroll-Triggered Animations**: Modern reveal transitions for all major sections using the `Intersection Observer API`.
- **🎨 Custom Design System**:
  - Vibrant Purple & Indigo Brand Palette.
  - Bespoke Gradient Scrollbar.
  - Fluid Typography (Responsive scaling for all headliners).

---

## 🚀 Technologies Used

- **HTML5**: Semantic tags and SVG integration.
- **CSS3 (Vanilla)**: Flexbox, Grid, Custom Properties (Variables), and Advanced Keyframe Animations.
- **JavaScript (ES6+)**: Custom slider logic, smooth scroll anchors, and scroll-reveal observers.
